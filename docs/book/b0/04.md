# 保持衣橱整洁的法则

## 30. 少买衣服

拥有太多的衣服，必然花费了更多的时间在挑选、购买、清洁、整理。
衣服越少，拥有自由的时间越多。

如果已经拥有了太多的衣服，一个简单的法则是扔掉。
如果特别想要拥有一件衣服，先想一下，需要扔掉哪几件。

简单舒适的生活，就是拥有的少一点，而不是多一点。
生活的幸福感来自于你拥有了多少自由的时间，而不是被物质所填满。

首先要舒适，其次才是美观。
